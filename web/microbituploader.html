Uploader for micro:bit

Tired of drag and dropping .hex files to your BBC micro:bit? Use the uploader to automate it!

Freely available on Windows.

Download (v0.6)

How does it work?

The uploader monitors your Downloads folder looking for any .hex file being download. When it detects a new .hex file, the uploader tries to copy it to any connected BBC micro:bit (it copies it to the MICROBIT drive). That's it!

System Requirements

  • Windows XP or later
  • .NET runtime 2.0 or higher
  • Browser: this application will work for any web browser that is supported by the BBC micro:bit

Installation Instructions

  • Download the Microsoft.Uploader.Microbit.zip file to your local computer.
  • Unzip the .zip file and copy the Microsoft.Uploader.exe file to your desktop.
  • Launch the Microsoft.Uploader.exe file before working on your BBC micro:bit
